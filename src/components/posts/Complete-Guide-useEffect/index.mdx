# مرحباً بالعالم

مرحباً بكم في أول منشور MDX لي! في هذا المنشور، سنغطي كيفية استخدام MDX مع Astro وكيفية استيراد محتوى MDX ديناميكيًا.

## المقدمة

يسمح لك MDX بكتابة محتوى Markdown ودمج مكونات JSX في نفس الملف. إنها أداة قوية تجمع بين بساطة Markdown ومرونة مكونات React.

## فوائد MDX

- **تكامل سلس**: كتابة محتوى Markdown ودمج مكونات React جنباً إلى جنب.
- **المحتوى الديناميكي**: تصدير البيانات من ملفات MDX واستخدامها بشكل ديناميكي في صفحاتك.
- **المكونات المخصصة**: استخدم مكونات React الخاصة بك أو المكتبات التابعة لجهات خارجية ضمن MDX.

### مثال على الكود

إليك كيفية عرض عنوان من MDX بشكل ديناميكي: 

```jsx

    <!doctype html>
    <html>
        <head>
            <meta charset="UTF-8" />
            <meta name="description" content="وصف Astro" />
            <meta name="viewport" content="width=device-width" />
            <link rel="icon" type="image/svg+xml" href="/favicon.svg" />
            <meta name="generator" content={Astro.generator} />
            <link rel="preconnect" href="https://fonts.googleapis.com" />
            <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
            <link href="https://fonts.googleapis.com/css2?family=Cairo:wght@200..1000&display=swap" rel="stylesheet" />
            <title>{title}</title>
        </head>
        <body class="bg-[#222831]">
            <main class="container mx-auto my-5" data-swup>
                <Header />
                <div class="flex items-start gap-7 mt-5">
                    <InformationSection />
                    <div class="transition-main">
                        <slot />
                    </div>
                </div>
            </main>
        </body>
    </html>

```
